## Display sizes
[Wikipedia](https://en.wikipedia.org/wiki/Display_size)

- Area

  $$S=W\cdot H$$

- Length of the diagonal

  $$D=\sqrt{W^2+H^2} =\sqrt{(W+H)^2-2S} =\sqrt{(1+k^2)W^2}$$

  - A square screen gives you the largest area for a given diagonal
  - $19'' + 23.5'' \approx 33'' $
